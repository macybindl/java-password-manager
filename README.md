# Java Password Manager

A secure CLI-based password manager written in Java. Stores credentials with AES encryption in a local file.

## Features
- Add, view, and search accounts
- AES-encrypted password storage
- Modular Java design with classes for Account, FileHandler, and Encryption

## Tech Stack
- Java
- AES Encryption
- File I/O

## Usage
1. Clone the repo
2. Compile and run `Main.java`
3. Add/view/search accounts via CLI
